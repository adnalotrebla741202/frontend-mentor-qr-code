<h1>Reto QR Code Component de FrontEnd Mentor</h1>
<h2>Descripción del proyecto</h2>
Este proyecto es una <b>tarjeta de componente QR (QR Code Component)</b> desarrollada como parte de un desafío de Frontend Mentor.
Su objetivo es <b>practicar la maquetación web y los estilos con HTML y CSS</b>, reproduciendo un diseño estático y centrado en la presentación visual.
<h2>Tecnologías utilizadas</h2>
<ul>
  <li>Usa <b>HTML5</b> para la estructura del contenido.</li>
  <li>Usa <b>CSS3</b> para los estilos, diseño visual y adaptabilidad a distintos dispositivos.</li>
  <li>Usa <b>Recursos de Frontend Mentor</b> los cuales son imágenes y recursos gráficos proporcionados para poder resolver el desafío.</li>
  <li>El proyecto incluye una media query para pantallas con un ancho máximo de 375px, asegurando que el diseño se adapte correctamente a teléfonos móviles.</li>
</ul>
<h2>Instrucciones de instalación y uso</h2>
<ul>
  <li>Para usar este código hay que ir a la pestaña Code | Download ZIP.</li>
  <li>Descomprimir el archivo zipeado, entrar a la carpeta creada y buscar el archivo <b>index.html</b>.</li>
  <li>Una vez localizado el archivo <b>index.html</b> le damos doble click a dicho archivo y se nos abrirá el proyecto HTML en nuestro navegador de Internet predeterminado.</li>
  <li>Si escogemos visualizar el diseño en dispositivos móviles el diseño se ajustara al tamaño de la pantalla del dispositivo de despliegue.</li>
</ul>
